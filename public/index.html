See what your AI actually remembers

Your Mem0 instance stores and retrieves memories. But is it working well? Engram instruments every operation — hit rates, score distributions, staleness, dead memories — so you stop guessing and start measuring.

Drop-in. One line changes.
# Before from mem0 import MemoryClient client = MemoryClient(api_key="...") # After from engramlabs import InstrumentedMem0 client = InstrumentedMem0(MemoryClient(api_key="...")) # Same API. Now you can see everything. results = client.search("dietary preferences", user_id="user123")
What you'll see
Hit Rate
% of searches returning relevant results above your score threshold.
Score Distribution
Are your top results actually good? See the spread across every search.
Dead Memories
Memories stored but never retrieved. Find the waste, clean it up.
Staleness
Age distribution of retrieved memories. Are you serving stale context?
Redundancy
Duplicate content detection. Stop storing the same memory twice.
Latency
p50, p95, p99 for every operation. Know when things slow down.
How it works
1
Wrap your Mem0 client. One import change. Works with both OSS and Platform.
2
Use Mem0 normally. Every add, search, update, and delete is silently instrumented.
3
Run engram report for an instant health check — or connect Grafana for live dashboards.
